- **Primaire**: #14b8a6 (Turquoise)

#### Boutons
- Primary (Turquoise avec hover)

### 📐 Breakpoints

- **Desktop**: > 1024px

## Instructions Figma

1. **Créer un nouveau fichier** : Portfolio Jacques Moriba
2. **Configurer les frames** selon les breakpoints
3. **Créer les styles** dans le panel "Styles"
4. **Créer les composants** avec auto-layout
5. **Utiliser les variants** pour les états hover
6. **Prototyper** les interactions principales
