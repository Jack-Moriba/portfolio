Toutes les maquettes incluent :

- Layout complet
- Navigation horizontale
- Multi-colonnes
- Interactions riches

### Tablette (768-1024px)
- Layout adapté
- Navigation collapsible
- 2-3 colonnes maximum
- Touch-optimisé

### Mobile (<768px)
- Single column
- Bottom navigation
- Full-width elements
- Gestures support
