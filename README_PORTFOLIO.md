# Jacques Moriba - Portfolio Professionnel

## 🌐 [Voir le Portfolio en Ligne](https://votre-portfolio.vercel.app)

Un portfolio moderne et interactif mettant en avant mes compétences en développement full-stack, avec une touche unique de design guinéen.

![Portfolio Preview](./public/images/preview.png)

## ✨ Fonctionnalités

### 🎨 Design & Expérience Utilisateur
- **Glassmorphism Design** - Interface moderne avec effets de transparence
- **Animations GSAP** - Animations fluides au scroll et interactions
- **Modèle 3D Interactif** - Personnage Three.js animé qui réagit à la souris
- **Effets Visuels Avancés** - Particules, glitch, hologrammes, cinématiques
- **Mode Sombre/Clair** - Thème adaptable avec transition fluide

### 🛠️ Technologies Utilisées
- **Frontend**: React 18 + TypeScript + Vite
- **Animations**: GSAP + ScrollTrigger
- **3D Graphics**: Three.js + React Three Fiber
- **Styling**: CSS3 avec variables dynamiques
- **UI Components**: Design system personnalisé

### 📱 Responsive & Accessibilité
- **Mobile-First** - Optimisé pour tous les appareils
- **Accessibilité WCAG 2.1** - Support lecteurs d'écran et navigation clavier
- **Performance Optimisée** - Lazy loading et code splitting
- **SEO Friendly** - Meta-tags et structure sémantique

### 🎯 Sections du Portfolio
1. **Hero** - Introduction avec modèle 3D
2. **About** - Présentation personnelle et parcours
3. **What I Do** - Services et compétences
4. **Career** - Timeline professionnelle
5. **Work** - Projets avec carrousel interactif
6. **Certifications** - Badges Cisco et formations
7. **Tech Stack** - Bento grid des technologies
8. **Services** - Offres de prestation
9. **Testimonials** - Témoignages clients
10. **Contact** - Formulaire et informations

### 🎮 Fonctionnalités Spéciales
- **Easter Egg Game** - Mini-jeu caché dans le portfolio
- **GuineaPay Demo** - Plateforme de paiement intégrée (démonstration)
- **Analytics Dashboard** - Statistiques de visite
- **Cinematic Intro** - Animation d'introduction immersive
- **Cursor Personnalisé** - Curseur avec effet de traînée

## 🛠️ Installation & Développement

### Prérequis
- Node.js 18+
- npm ou yarn

### Installation
```bash
# Cloner le repository
git clone https://github.com/Jack-Moriba/portfolio.git
cd portfolio

# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev
```

### Build Production
```bash
# Créer le build de production
npm run build

# Prévisualiser le build
npm run preview
```

### Tests
```bash
# Linter
npm run lint

# Type checking
npx tsc --noEmit
```
